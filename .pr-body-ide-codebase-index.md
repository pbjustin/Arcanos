## Summary
Add workspace file and docs so IDEs (Cursor, VS Code, GitHub Codespaces) and cloud agents (Codex) can index and use the Arcanos codebase with full context.

## Changes
- **Arcanos.code-workspace** – Portable workspace file (single folder `.`) so any clone can open it; codebase indexing targets Arcanos only.
- **.cursorignore** – Keeps index focused on source (excludes node_modules, venvs, __pycache__, dist, logs, secrets).
- **CODEBASE_INDEX.md** – Short map for AI/agents: entry points, key dirs, where to find API contract, OpenAI, DB.
- **docs/IDE_AND_AGENTS.md** – How to open the workspace for Cursor/VS Code/Codespaces/cloud agents; .cursorignore template.
- **README.md** – New section "IDE and cloud agents" with link to workspace file and CODEBASE_INDEX.

## Usage
Open **Arcanos.code-workspace** (repo root) via File → Open Workspace from File so semantic search and rules (.cursorrules, AGENTS.md) apply to Arcanos only.

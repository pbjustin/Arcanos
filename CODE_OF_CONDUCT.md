# Code of Conduct

## Overview
We are committed to a respectful, inclusive, and technically constructive community across issues, pull requests, code review, and discussions.

## Prerequisites
This policy applies to everyone participating in the Arcanos project spaces.

## Setup
Expected behavior:
- Be respectful and professional.
- Focus feedback on code and behavior, not individuals.
- Help others learn through clear and actionable comments.

Unacceptable behavior:
- Harassment, discrimination, threats, or personal attacks.
- Publishing private information without consent.
- Deliberately harmful security or supply-chain behavior.

## Configuration
Reporting process:
- Use GitHub's Code of Conduct issue template for non-sensitive cases.
- For sensitive incidents, use private maintainer contact channels.
- Maintainers will acknowledge reports promptly and apply proportional enforcement.

## Run locally
Not applicable.

## Deploy (Railway)
Not applicable.

## Troubleshooting
If a report is not acknowledged in a reasonable window, open a follow-up issue and request maintainer escalation.
